# Claims Ledger

This file separates what the repository currently demonstrates from what remains preliminary or hypothetical.

## False vs Real Unpredictability

- `False unpredictability` in this repo means the surprise is largely created by the observation layer: coarse or thresholded metrics make smooth internal progress look abrupt.
- `Real unpredictability` in this repo means the surprise remains after metric controls: the forecast misses, precursor mechanism signals stay weak, or the current theory does not explain the case.

The current benchmark package supports `false unpredictability` on the periodic modular case and `real unpredictability` on the sparse relational stress case. See [`artifacts/minimal_suite/suite_summary.json`](/Users/hifi/Capability-Cartography-Layer-4/artifacts/minimal_suite/suite_summary.json).
